# Release Notes: v0.1.0-playtest

Date: 2026-07-10

This playtest release is intended for friends, community testers, caregivers, and clinicians to try the prototype and share structured feedback.

## What is new

- Added a baseline playable experience across three games:
  - Recall
  - Pair Match
  - Sequence Recall
- Added resident history view with score trends and session records.
- Added a Playtest Feedback section on the home page to capture notes and ease-of-use ratings.
- Improved accessibility defaults:
  - Larger base text
  - Minimum 44px interactive controls
  - Stronger keyboard focus states
  - Reduced-motion support
- Added tester onboarding guide in TESTER-QUICKSTART.md.

## Notes

- Prototype data is currently stored in local browser storage.
- This tool is for engagement and playtesting only, not diagnosis.

## Request for testers

Please share:

- What felt easiest or most enjoyable
- What felt confusing or too fast
- One change that would make the game better for first-time users
