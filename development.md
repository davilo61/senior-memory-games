# Development Notes

### Study Phase

You see:

```
🍎 Apple
🐶 Dog
🪑 Chair
🕰️ Clock
🌷 Flower
🎸 Guitar
```
                 
and the button:

```
Start Recall Phase
```

### Recall Phase

When clicked:

```
Which items did you see? b                
```

The original cards disappear.

Instead you'll see 10 choices:

```
🍎 Apple
🐶 Dog
🪑 Chair
🕰️ Clock
🌷 Flower
🎸 Guitar
🚗 Car
📚 Book
👟 Shoe
💡 Lamp
```

You can only select **6 items** (the game will not allow more than 6 selections). The UI shows how many selections you have remaining.

When you click one:

- it turns blue

- clicking again deselects it

At that point you'll have your first real gameplay mechanic.

Once you get that working, we'll add:

1. **Submit Answers**

2. **Score Calculation**

3. **Results Screen**

4. **Play Again**
