# Tester Quick Start

Use this guide when sharing the game with friends, residents, caregivers, or interested staff.

## 1. Start the app locally

```bash
npm install
npm run dev
```

Open http://localhost:3000 in a browser.

## 2. Run one full test session

1. Add a resident on the home page.
2. Play all three games at least once:
   - Recall
   - Pair Match
   - Sequence Recall
3. Visit History for that resident.
4. Return to home and submit a note in Playtest Feedback.

## 3. What to observe

- Could the user understand how to begin without help?
- Which screen felt confusing or too fast?
- Were text and buttons easy to read and tap?
- Did they look interested after the first round?

## 4. Suggested feedback format

When adding feedback notes, include:

- Who tested (friend, resident, caregiver, clinician, developer)
- What worked
- What was confusing
- One specific improvement request

## 5. Known limits for this playtest

- Data is stored locally in browser storage for this prototype.
- This is not a diagnostic or medical assessment tool.
- Session data may not sync across devices.
